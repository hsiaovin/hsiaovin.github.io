LevelDB源码分析

LevelDB是由Jeff Dean和Sanjay Ghemawat发起的一套用于持久化数据的高性能类库，key-value储存系统，源码短小精悍，代码风格极好。 需要注意的是，LevelDB存在一定的限制，需要用户自行实现server，且只有单进程能访问数据库。

LevelDB源码目录

  • db // DB逻辑
  • helpers //
  • include // LevelDB的接口
  • port // 操作系统的相关接口
  • table // 表存储结构
  • util // 公用组件

LevelDB接口

  • c.h // LevelDB的c的接口
  • cache.h // 缓存接口，默认使用LRU算法
  • comparator.h // 比较器接口，用于比较Key的大小
  • db.h // DB的主要接口
  • dumpfile.h //
  • env.h // 考虑到移植以及灵活性，LevelBD将系统相关的处理抽象成Env
  • filter_policy.h // filter策略，提供BloomFilter的实现
  • iterator.h // 数据库的遍历接口
  • options.h // 控制数据库行为和性能的配置选项
  • slice.h // 为操作数据的方便，将数据和长度包装成Slice使用，直接操控指针避免不必要的数据拷贝。
  • status.h // 错误信息状态的接口
  • table.h // immutable persistent sortedtable
  • table_builder.h // 创建table的接口
  • write_batch.h // 批量（原子）修改功能的接口，类似MySQL中事务
